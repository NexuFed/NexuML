# Learn NexuML

A progressive learning path for ML researchers. Work through these pages in order, or jump to the topic you need.

## Recommended path

1. **[Mental model](mental-model.md)** — the `ScenarioSpec` lifecycle from declaration to trained model.
2. **[Coming from Lightning](from-lightning.md)** — how NexuML maps onto PyTorch Lightning if you already know it.
3. **[Scenarios](scenarios.md)** — anatomy of a `ScenarioSpec`: data, pipeline, training, evaluation, and export.
4. **[Decorators and discovery](decorators-and-discovery.md)** — `@scenario`, `@layer`, `@data_source`, `@eval_algorithm`, and how NexuML finds them.
5. **[CIFAR ResNet walkthrough](cifar-resnet-walkthrough.md)** — read through a real scenario to see how all the pieces fit together.

## After the learning path

Once you understand the mental model and scenario structure, the [how-to guides](../how-to/index.md) give you task-focused recipes for every major workflow.

The [reference section](../reference/index.md) provides generated API docs, CLI flag reference, and compact decorator/registry tables.
